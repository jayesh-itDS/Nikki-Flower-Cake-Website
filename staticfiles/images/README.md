# Placeholder Images

This folder contains placeholder images for development purposes. Actual product images will be uploaded to this directory in production.

## Image Guidelines

- All product images should be high resolution (minimum 800x800 pixels)
- Maintain consistent aspect ratios
- Optimize for web (compress without losing quality)
- Use descriptive filenames

## Current Placeholders

- hero-floral.jpg: Main hero section image
- category-*.jpg: Category browsing images
- product-*.jpg: Individual product images
- placeholder-product.jpg: Generic product placeholder
